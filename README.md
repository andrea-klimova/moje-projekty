- **POZNAMKY.md** — co jsi změnila, proč, co nefungovalo

## Jak na to s Claude Code

Na začátku každé práce řekni Claude Code:
> "Přečti si soubor PLAN.md a PROMPT.md v složce [název projektu] a pokračuj podle toho."

Tím nemusíš pokaždé vysvětlovat kontext od začátku.
