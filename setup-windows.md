# Návod: První nastavení na Windows

## 1. Založení GitHub účtu
1. Jdi na https://github.com
2. Klikni "Sign up"
3. Zadej e-mail, heslo, uživatelské jméno
4. Potvrď e-mail

## 2. Anthropic API klíč
1. Jdi na https://console.anthropic.com
2. Přihlas se (jiný účet než Claude.ai — nebo stejný, ale jiná sekce)
3. Vlevo klikni "API Keys"
4. Klikni "Create Key"
5. Klíč si ulož — zobrazí se jen jednou!
6. POZOR: klíč nikdy nedávej do kódu, jen do GitHub Secrets

## 3. Gmail App Password
1. Jdi na https://myaccount.google.com
2. Bezpečnost → Dvoufázové ověření (musí být zapnuté)
3. Bezpečnost → Hesla aplikací
4. Vyber "Jiná" → napiš "blog-citecka"
5. Ulož si vygenerované heslo (16 znaků)

## 4. Instalace Claude Code na Windows
1. Jdi na https://docs.claude.ai
2. Stáhni Claude Code pro Windows
3. Nainstaluj
4. Otevři terminál (Win + R → cmd nebo PowerShell)
5. Přihlas se svým Claude účtem
